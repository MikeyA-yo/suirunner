# Sui-runner Documentation

This is the documentation site for Sui-runner, built with [Starlight](https://starlight.astro.build).

## 🚀 Get Started

1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`

## 📁 Project Structure

- `src/content/docs/` — All of your documentation content.
- `astro.config.mjs` — Starlight integration settings.

## 🧞 Commands

| Command           | Action                                       |
| :---------------- | :------------------------------------------- |
| `npm install`     | Installs dependencies                        |
| `npm run dev`     | Starts local dev server at `localhost:4321`  |
| `npm run build`   | Build your production site to `./dist/`      |
| `npm run preview` | Preview your build locally                   |
