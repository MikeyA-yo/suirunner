Sui-runner ships with a live Terminal User Interface (TUI) dashboard that shows your active Sui environment at a glance.

## Usage

```bash
sui-runner dashboard
```

The dashboard opens in your terminal and displays real-time information about your Sui development environment.

## What's Displayed

- **Active address** — the wallet address currently selected in the Sui CLI
- **Network** — which network you're connected to (mainnet, testnet, devnet, or localnet)
- **Project info** — details about the Move package in the current directory

## Exit

Press `q` or `Ctrl+C` to close the dashboard and return to your shell.
